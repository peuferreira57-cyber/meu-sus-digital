Sistema para facilitar o acesso a medicamentos do SUS
# Meu SUS Digital

Projeto desenvolvido para o Programa Centelha com o objetivo de melhorar o acesso a medicamentos no SUS.

## 💡 Sobre o projeto

O sistema propõe uma solução para ajudar usuários a encontrar medicamentos disponíveis em unidades de saúde próximas, reduzindo deslocamentos desnecessários.

## ⚙️ Funcionalidades

* Busca de medicamentos
* Localização de unidades de saúde próximas
* Informação de estoque (disponível, baixo ou em falta)

## 🛠️ Tecnologias

* Python / C# (ajuste conforme o que você usou)
* Lógica de programação
* Conceitos de API e geolocalização

## 🎯 Objetivo

Melhorar a eficiência logística e a transparência no acesso a medicamentos públicos.
